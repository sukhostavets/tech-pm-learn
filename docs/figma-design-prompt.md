# Figma Make Design Prompt: Tech Stable Academy

## Project Overview
Create a web application design for "Tech Stable Academy" - a gamified learning platform for Technical Product Management skills themed around horse stable management. The platform transforms complex tech concepts into an engaging stable management simulation.

## Visual Design Direction

### Color Palette
**Primary Colors:**
- **Soft Pink (#FFB6C1)**: Main brand color, primary UI elements, headers
- **Rose Pink (#FF69B4)**: Accents, buttons, highlights
- **Blush Pink (#FFC0CB)**: Backgrounds, cards, secondary elements
- **Dusty Rose (#C78C94)**: Borders, subtle accents

**Secondary Colors:**
- **Rich Brown (#8B4513)**: Navigation bars, footers, stable/wood textures
- **Saddle Brown (#654321)**: Text, icons, important elements
- **Tan/Beige (#D2B48C)**: Neutral backgrounds, paper textures
- **Cream (#FFFDD0)**: Light backgrounds, content areas

**Accent Colors:**
- **Gold (#FFD700)**: Achievement badges, XP indicators, premium features
- **Mint Green (#98FF98)**: Success states, completed items
- **Soft Coral (#FF7F50)**: Active states, notifications

### Design Style
- **Modern but Warm**: Clean interface with cozy, welcoming elements
- **Playful Professional**: Balances serious learning content with fun gamification
- **Illustrations**: Hand-drawn style horses, stables, and farm elements
- **Typography**: 
  - Headers: Rounded, friendly sans-serif (like Quicksand or Poppins)
  - Body: Clean, readable (Inter or Open Sans)
  - Accent: Handwritten style for gamification elements

## Core Pages to Design

### 1. Landing Page / Home
**Hero Section:**
- Illustration of a beautiful pink-tinted stable with horses
- Headline: "Build Your Tech Skills, Grow Your Stable"
- Subheading: "Master Technical Product Management through stable management simulation"
- CTA Button: "Start Your Journey" (pink with brown text)

**Features Section:**
- 3-4 cards showing key features
- Icons: horseshoe (progress tracking), trophy (achievements), book (learning), community (riders)

**Visual Elements:**
- Soft pink gradient background
- Brown wooden fence dividers between sections
- Scattered horseshoe and flower decorations

### 2. Dashboard (Main Hub)
**Layout:**
- Top Navigation: Brown bar with logo, profile picture, notification bell
- Welcome Banner: "Welcome back to [User's Stable Name]!"
- XP Bar: Pink progress bar with horse icon that moves as you progress
- Currency Display: "Hay Coins" (your learning currency) with wheat icon

**Main Content Area:**
- **Your Stable Overview**: 
  - Visual stable with 7 horse stalls (one per milestone)
  - Each stall represents a learning milestone
  - Locked stalls shown in greyscale with padlock
  - Active stall highlighted with pink glow
  - Completed stalls show healthy, happy horses

- **Current Quest Card**: 
  - Pink card with brown border
  - Shows current lesson/activity
  - Progress indicator
  - "Continue Learning" button

- **Daily Challenges Box**:
  - Checklist with horseshoe checkboxes
  - Reward indicators

**Sidebar:**
- Achievement badges display
- Streak calendar (horseshoes marking days)
- Leaderboard preview

### 3. Milestone Map (Learning Path)
**Visual Concept:**
- Illustrated path through stable grounds
- 7 locations representing milestones:
  1. **Training Arena** - SDLC/DevOps (brown sand arena)
  2. **Feed Storage** - Data/SQL (pink barn with data charts)
  3. **Pasture Fields** - Cloud Infrastructure (open sky, clouds)
  4. **Stable Complex** - System Design (architectural building)
  5. **Riding Trails** - API/Platform (connected paths)
  6. **Breeding Barn** - Machine Learning (genetics/growth)
  7. **Show Arena** - GenAI/LLMs (spotlight, performance)

**Interaction:**
- Click each location to see milestone details
- Path lights up as you progress
- Locked areas shown with fence barriers

### 4. Lesson Interface
**Layout:**
- **Left Sidebar (20%)**: 
  - Lesson navigation tree
  - Progress indicators (pink checkmarks)
  
- **Main Content (60%)**:
  - Clean reading area with cream background
  - Embedded visuals and diagrams
  - Brown section dividers
  - Interactive elements highlighted in pink
  
- **Right Sidebar (20%)**:
  - Quick definitions panel
  - Bookmark feature
  - Notes section
  - Mini achievement tracker

**Bottom Fixed Bar:**
- Pink progress bar for current lesson
- "Previous" / "Next" buttons (brown)
- "Take Quiz" button when ready (pink, prominent)

### 5. Quiz/Assessment Page
**Visual Style:**
- Question card: White/cream with pink border
- Multiple choice options: Brown outlined buttons that turn pink when selected
- Correct answers: Mint green highlight with checkmark
- Incorrect answers: Soft coral with explanation popup

**Gamification:**
- Timer with horseshoe icon counting down
- XP preview showing potential earnings
- Horse character reacting to correct/incorrect answers (animated)

### 6. Mini-Game: Term Hangman
**Design:**
- Hanging horseshoe instead of traditional hangman
- Wrong guesses add pieces to fallen rider (gentle, not morbid)
- Letter buttons styled as hay bales (brown, turn pink when clicked)
- Background: Stable interior with pink-painted walls
- Success animation: Horse neighing and prancing

### 7. Project Submission Page
**Layout:**
- Large upload area with pink dashed border
- Drag-and-drop zone illustrated as "Delivery Cart"
- File preview cards with brown headers
- Rubric checklist with horseshoe checkboxes
- "Submit Project" button (large, pink, prominent)

**Progress Indicator:**
- Shows deliverable checklist
- Each item is a horseshoe that fills with pink when completed

### 8. Profile / Stable Management
**Hero Section:**
- Banner showing user's stable with all their horses
- Stable name (editable)
- User avatar (can be customized with rider gear)

**Stats Dashboard:**
- Pink cards with brown borders showing:
  - Total XP (with level indicator)
  - Hay Coins balance
  - Streak counter
  - Completion percentage
  
**Achievement Gallery:**
- Badge display in horseshoe shapes
- Tooltips explaining each achievement
- Locked achievements shown in greyscale

**Horse Collection:**
- Each completed milestone "unlocks" a horse
- Different breeds for different milestones
- Horses have stats based on quiz scores
- Can name and customize horses

## UI Components to Design

### Buttons
- **Primary**: Pink background, brown text, rounded corners
- **Secondary**: Brown outline, pink text, rounded corners
- **Disabled**: Grey with lower opacity
- **Hover States**: Subtle shadow, slight color shift

### Cards
- White/cream background
- Pink or brown top border (3px)
- Soft shadow
- Rounded corners (8px)

### Progress Bars
- Pink fill, cream background
- Brown border
- Horse icon that moves along the bar

### Icons
- Custom icon set combining tech symbols with stable elements:
  - Database = Feed bucket
  - API = Riding trail connections
  - Cloud = Clouds above pasture
  - Bug/Error = Horseshoe with crack
  - Success = Rosette ribbon

### Modals/Popups
- Cream background with pink border
- Brown "X" close button
- Celebration animations for achievements (confetti in pink/brown)

### Navigation
- Top nav: Brown bar, pink hover states
- Breadcrumbs: Brown with pink separators (horseshoe icons)
- Footer: Brown with pink accents

## Interactive Elements

### Animations
- Horse trotting animation for loading states
- Sparkle effects when earning XP (pink/gold particles)
- Horseshoe flip for completed tasks
- Gentle bounce on button hovers
- Stable door opening for unlocking new content

### Tooltips
- Cream background, brown text
- Pink arrow pointer
- Appears on hover with smooth fade

### Notifications
- Toast messages styled as "Stable Signs" (wooden boards)
- Pink ribbon banner for achievements
- Brown border, cream background

## Responsive Considerations
- Mobile: Stack stable stalls vertically
- Tablet: 2-column layout for dashboard
- Desktop: Full 3-column layout with sidebars

## Accessibility Notes
- Ensure pink/brown contrast meets WCAG AA standards
- Provide alternative text for all horse illustrations
- Keyboard navigation for all interactive elements
- Screen reader friendly labels

## Mood and Tone
- **Welcoming**: Like entering a well-maintained, cozy stable
- **Encouraging**: Progress feels rewarding, mistakes are learning opportunities
- **Professional but Playful**: Serious content delivered in an engaging way
- **Feminine but Universal**: Pink/brown palette appeals to target user but remains inclusive
- **Growth-Oriented**: Visual metaphor of caring for horses mirrors skill development

---

**Final Note**: The stable/horse theme should enhance the learning experience, not distract from it. Technical content remains clear and accessible while the gamification layer adds motivation and joy to the learning journey.
